# project-storage-products
مشروع Inventory Management System هو نظام بسيط لإدارة المخزون تم تطويره باستخدام لغة ++C، ويهدف إلى تنظيم المنتجات داخل المخزن من خلال إضافة المنتجات، تعديل الكميات، عرض المنتجات، البحث عن منتج، وحساب إجمالي الكميات باستخدام Recursion. تم استخدام مفاهيم برمجية مهمة مثل Struct و Arrays و Functions لتنظيم البيانات وتسهيل إدارة النظام.
